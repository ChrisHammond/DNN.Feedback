DotNetNuke® Feedback Module v. 06.02.00 -- RELEASE NOTES

Requirements: DotNetNuke version 06.02.00 or later, ASP.Net 3.5 SP1 or later.

Full details on the changes can be found in great detail at http://dnnfeedback.codeplex.com/workitem/list/basic.

This release brings some enhancements such as integrating to DNN Notifications service as well as fixing all known bugs.

Bug Fixes

  • Removed error in Get_Feedback sproc which was restricting the ability to print feedback.
  • Removed error in Get_Feedback sproc which was causign Azure install to fail.

Enhancements

  • Integrate to DNN 6.2.0 Notification service for Moderator notifications.
  • Added setting to enable the rejection of duplicate submissions.
  • Multiple email addresses may now be used in Category value as Send To addresses.
  • Added configureable scheduled process to delete old Feedback.

Changes

  • Changed to use IconKey api for print icon which was not available in DNN 6.1.2
