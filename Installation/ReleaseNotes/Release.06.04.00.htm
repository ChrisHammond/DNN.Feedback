DotNetNuke® Feedback Module v. 06.04.00 -- RELEASE NOTES

Requirements: DotNetNuke version 06.02.01 or later, ASP.Net 3.5 SP1 or later.

Full details on the changes can be found in great detail at http://dnnfeedback.codeplex.com/workitem/list/basic.

This release brings a number of enhancements, the major one being the inclusion of Akismet support.

Enhancements

  • Added ability to disable sending of mail to receipients until feedback is published.
  • Added setting to allow configuration of page size in Moderation/Management view.

Changes

  • Changed margins to be percentages from fixed with to improve usability on small screen sizes.
  • Code tidied up and improved for Azure installation.

Bug Fixes

  • Included workaround for saving Sent To Roles to handle bug introduced in DNN 6.2.0.
